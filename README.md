# Janya Vyas — Portfolio v3.0

Light, interactive, refokus-inspired portfolio. Built with React + Three.js + Framer Motion.

## Quick Start
```bash
npm install
npm run dev
# → http://localhost:5173
```

## Build & Deploy
```bash
npm run build
# Vercel: npx vercel
# GitHub Pages: npm run build && gh-pages -d dist
```

## Key Features
- **Light theme** — warm `#F5F3EE` paper background
- **Custom cursor** — smooth lagging ring, morphs on hover
- **Three.js hero** — light wireframe icosahedron + orbit lines + particles
- **Expandable experience cards** — click to reveal achievements (accordion)
- **Skill tabs** — click category to filter pills with animated transitions
- **Magnetic CTAs** — buttons subtly follow the cursor
- **Framer Motion** — stagger reveals, layout animations, spring physics
- **Fonts** — Clash Display + Cabinet Grotesk + JetBrains Mono
- **Scroll progress bar** — thin blue line at top
- **Active nav highlighting** — smooth pill transition between sections
- **Print-ready** — clean PDF export via browser print

## Customise
All content in `src/data/content.js` — edit and save.

## Tech Stack
React 18 · Vite · Tailwind CSS · Framer Motion · Three.js + R3F · Lucide React
